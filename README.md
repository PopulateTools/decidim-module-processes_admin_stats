# Decidim::ProcessesAdminStats

A Decidim module to include statistics of participatory processes in their admin panel.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-processes_admin_stats"
```

And then execute:

```bash
bundle
```
